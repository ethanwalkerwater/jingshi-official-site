## 二、设计系统（已确定方向）

**设计方向：常春藤传承 × 哈佛式严谨 × 现代无衬线**

参考哈佛大学官网的设计语言（白底 + 高对比深蓝、严格网格、模块化卡片、大量留白、几乎零装饰、权威严谨），结合菁仕自有品牌色，并采用现代字体。

### 配色（取自 logo：海军蓝 + 金）

| 角色 | 色值 |
| --- | --- |
| 海军蓝 主色 | `#14274a` |
| 海军蓝 深 | `#0e1d38` |
| 金 主点缀 | `#b88a3d` |
| 金 深 | `#9a722d` |
| 背景 近白 | `#fbfaf7` |
| 纯白卡片 | `#ffffff` |
| 正文墨色 | `#141c2e` |
| 次要文字 | `#5a6377` |
| 分隔细线 | `rgba(20,39,74,.13)` |

> 配色保留菁仕自有的海军蓝 + 金（logo 色），**不采用哈佛的 crimson 红**。

### 字体（现代无衬线为主）

- 中文标题 / 正文：**Noto Sans SC**（标题用 900 重磅字重，正文 400）
- 英文 / 数字 / 标签：**Space Grotesk**（现代 grotesk）
- 弃用旧式衬线（Noto Serif SC / Cormorant Garamond），以获得更现代的观感

### Hero 设计原则

- **图片驱动、文字精简**：右侧真实大图（毕业季 / 校园 / 课堂），左侧仅核心标语 + 一句话 + 一个主按钮 + 真实名师头像条
- 大图当前为占位图，待替换为菁仕自有的校区 / 课堂 / 师生实拍

### 通用设计原则（反模板 / 反 AI slop）

- 严格网格、模块化卡片、细线分隔、大量留白
- 金色仅作克制点缀（导航下划线、序号、评分角标、按钮）
- 哈佛招牌「In Focus」资讯带：用于展示申请季 offer 捷报
- 衬线/装饰最小化，高对比、权威、严谨

## 五、课程体系内容（四大板块）

> 每门课只做简短介绍（是什么 + 基础背景 + 所需知识体系）。

### ① 国际课程体系（Curriculum）
- 一句话：主流国际学校课程的同步与培优，打牢学术主线，对接全球名校申请。
- 适合人群：小升初至高中 · 国际 / 双语学校在读

| 课程 | 简介 |
| --- | --- |
| **IGCSE**（International GCSE） | 英国体系下 14–16 岁的国际中学课程，是 A-Level / IB 的衔接基础。覆盖数学、科学、英语与人文等学科，帮助学生在升入高中课程前建立扎实的学科底子与英文学术表达能力。 |
| **A-Level**（GCE Advanced Level） | 英国高中课程与升学的「黄金标准」，通常选修 3–4 门。以学科深度专精见长，成绩直接对接英国及全球大学申请，适合目标明确、希望发挥学科优势的学生。 |
| **IB**（International Baccalaureate） | 国际文凭大学预科项目，需修读六大学科组并完成 TOK、EE 与 CAS。强调全面发展与批判性思维，适合学科均衡、学有余力、面向全球顶尖院校的学生。 |
| **AP**（Advanced Placement） | 美国大学先修课程，可在高中阶段修读大学难度的单科考试。高分有助于体现学术能力、增强美本申请竞争力，并可能兑换大学学分。 |
| **Top-up Degree**（本科学位衔接） | 帮助已完成大专或部分本科的学生衔接英国等海外院校的最后一年，取得正式学士学位。 |

### ② 标化语言考试（Standardized）
- 一句话：海外院校申请的硬通货，从基础到冲刺，系统提分。
- 适合人群：初中至大学 · 准备海外升学的学生

| --- | --- |

| **IELTS**（雅思） | 全球广泛认可的英语水平考试，分学术类与培训类。英国、澳洲、加拿大等院校申请的主流语言门槛。 |
| **SAT**（美国学术能力测试） | 美国本科入学的标准化学术测试，考查阅读、文法与数学。是冲刺美国名校的重要标化成绩。 |
| **ACT**（美国大学入学考试） | 与 SAT 并列的美国本科入学考试，增设科学推理部分，更侧重答题节奏与综合学科能力。 |
